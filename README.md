# Donut Boilerplate

[![CircleCI](https://circleci.com/gh/kirill-krasuk/new_boilerplate.svg?style=svg)](https://circleci.com/gh/kirill-krasuk/new_boilerplate.svg)

## How To Start
___

#### You need to install dependencies

```
    npm i
```

#### Copy Dotenv file

```
    cp .env.example .env
```


## Available Scripts
---

Start developing
### `npm start`

Start production
### `npm run start:prod`

Run test
### `npm run test`

Use storybook
### `npm run storybook`
